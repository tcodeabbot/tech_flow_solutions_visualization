# TechFlow Solutions Dashboard

A modern, interactive Data Visualization Dashboard built with **Streamlit** and **Plotly**. 
This dashboard is designed to provide responsive, high-level business analytics covering both executive metrics and operational insights.

## Features

- **Light & Dark Mode**: Seamlessly toggle between Light and Dark themes dynamically.
- **Executive Analytics**: High-level KPIs including Total Revenue, Gross Profit, and Profit Margin. Visualizes Revenue Trends and regional breakdowns.
- **Operations Engine**: Deep-dive into Average Return Rates, Profit Per Unit, and Customer Satisfaction analytics.
- **Dynamic Filtering**: Interactive multidimensional filtering by Region and Product Category.
- **Responsive Layout**: Specially constrained ultra-wide views ensure that data looks perfect on screens of all sizes.

## Installation & Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/tcodeabbot/tech_flow_solutions_visualization.git
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the Streamlit application:
   ```bash
   streamlit run app.py
   ```
